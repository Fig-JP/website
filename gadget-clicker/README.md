# 端末クリッカー — 遊び方

端末を増やして計算資源を育て、インフラやネットワークを運用しながら巨大なガジェット帝国を作る放置・運用ゲームです。

▶ [端末クリッカーを遊ぶ](https://fig-jp.github.io/website/gadget-clicker/)

## ゲームの目的

画面中央の **CHIP** や端末から `PP（Processing Point）` を生産します。PPを使って端末・設備・強化を増やし、最初は **周回処理量100万PP** と「システムリセット」を目指しましょう。

リセット後は、獲得した最適化コアや恒久強化を引き継いで、前の周回より速く成長できます。長期的には研究、特別端末、自作PC、世界共同計算プロジェクトなどの完成が目標です。

## 最初の10分

1. **CHIPをタップする**<br>
   まずは手動でPPを作ります。10回タップすると最初の目標を達成できます。
2. **電卓を買う**<br>
   15 PP貯まったら、端末一覧から電卓を導入します。端末は何もしなくても毎秒PPを生産します。
3. **端末を増やす**<br>
   電卓からスマートフォン、ノートPCへと買い進めます。購入数ボタンでまとめ買いもできます。
4. **やることリストを確認する**<br>
   ゲーム画面上部のリストが、おすすめの順番を16段階で案内します。項目を押すと対応する機能へ移動します。
5. **メンテナンスを成功させる**<br>
   タイピングまたは配線チャレンジで整備パーツを入手します。整備パーツは研究やオーバードライブに使用します。
6. **電源と冷却を増強する**<br>
   端末が増えると電力消費と発熱が増えます。「運用」タブで稼働効率を確認し、容量が不足する前に設備を強化します。

迷ったら、画面上部の **NEXT** と **やることリスト** に従えば進められます。

## 基本の成長ループ

`タップでPPを作る → 端末を買う → 自動生産を増やす → 設備と強化へ投資する → より高性能な端末を買う` の繰り返しが基本です。

- **タップ強化**: CHIPを1回押したときのPPを増やします。
- **生産連携強化**: 毎秒生産量の一部をタップ威力へ加えます。
- **端末レベル**: シリコンクリスタルを消費し、その種類の端末生産を永続的に15%ずつ強化します。
- **シナジー**: 端末の所持構成を工夫すると、自動的に生産倍率が発動します。
- **ゴールデンパケット**: ときどき現れるパケットを押すと、一時強化や資源を獲得できます。
- **オフライン進行**: 画面を閉じている間の生産は、次回起動時にまとめて受け取れます。

## 画面の見方

### ゲーム

CHIP、端末購入、タップ強化、メンテナンス、シリコンクリスタル、システムリセットがあります。普段の生産と成長はこの画面で行います。

### 実績

プレイ状況に応じて実績が解除されます。一部は条件が伏せられた隠し実績です。解除数に応じて新しいテーマも使えるようになります。

### 研究

メンテナンスで手に入れた整備パーツを消費し、生産・タップ・購入価格などの恒久効果を解放します。前提となる研究から順に進めます。

### 運用

端末群を「ただ増やす」だけではなく、効率よく動かすための管理画面です。各カード右上の **?** を押すと、その機能の簡単な説明が表示されます。

### 設定

効果音・BGM・テーマ・セーブデータを管理します。機種変更前や大きな操作の前には、エクスポートでバックアップを保存しておくと安心です。

## 運用タブの機能

| 機能 | 役割 | 覚えておくこと |
| --- | --- | --- |
| 電力・発熱・冷却 | 端末を動かす基盤 | 容量を超えると生産効率が低下します |
| ネットワーク構成 | 生産・タップ・設備負荷を調整 | 所持端末によって構成が解放されます |
| ジョブ | PPを使った時間制の計算 | 完了時にPPや恒久効果などを獲得します |
| OSチャンネル | 安定性と速度を調整 | 高速なOSほど電力と発熱が増えます |
| AIオペレーター | 設備や端末を自動調整 | インフラ・拡張・均等化から方針を選びます |
| 端末の世代交代 | 古い端末を再利用 | 回路部品や恒久効果を獲得できます |
| 自作PC | 回路部品で専用機を作成 | 4種類のパーツをLv.1にすると稼働します |
| 特別端末 | 固有特性を持つ愛機 | 同じ端末を10・25・50台にすると獲得します |
| ラック配置 | 3×4の盤面で設備を配置 | 隣接関係によって生産・冷却・防衛が変わります |
| パイプライン | データを収集から推論まで処理 | 前工程用の端末が不足すると後工程が止まります |
| 運営組織 | その周回の得意分野を選択 | 次のシステムリセットまで変更できません |
| パケットルーティング | 順番入力ミニゲーム | 成功するとPPと回路部品を獲得します |
| サイバー防衛 | 時間制の侵入イベント | ログ解析で脅威値を0にします |
| 世界共同計算 | PPを長期プロジェクトへ提供 | 完成報酬は恒久的に適用されます |
| 運用ログ | 操作や障害の履歴 | インシデントでは対応方針を1つ選びます |

## おすすめの進行順

ゲーム内の「やることリスト」は、次の5段階で進みます。

1. **起動** — CHIPをタップし、最初の端末を買い、合計10台へ増やす
2. **基盤整備** — メンテナンス、研究、電源・冷却、ジョブを体験する
3. **高度運用** — 端末の再利用、自作PC、特別端末、ラック配置を進める
4. **ネットワーク** — 組織を選び、パイプラインとサイバー防衛を運用する
5. **長期目標** — 世界プロジェクトを完成させ、システムリセットへ進む

リストの達成状況はシステムリセット後も残ります。将来の項目を先に達成しても問題ありません。

## ミッションコントロール

ゲーム画面右側の **帝国キャンペーン** は、機能の羅列ではなく「何のために成長するのか」を示す5章の目標です。各章の条件をすべて達成したら、報酬を自分で受け取って次の章へ進みます。

その下の **運用契約** は、タップ・端末導入・メンテナンスを組み合わせた短い目標です。時間制限やログイン必須の要素はなく、3件を達成すると少し難しい次の契約セットを好きなタイミングで受注できます。

キャンペーンと契約の報酬は、PPだけでなく整備パーツ、回路部品、クリスタル、最適化コアにも広がります。新しい機能に触る理由と、次の操作を決める手がかりとして使ってください。

## システムリセット

現在の周回処理量が100万PPに到達すると、最初のシステムリセットが可能になります。

### リセットで得られるもの

- 生産倍率になる **最適化コア**
- 生産・タップ・端末価格から選ぶ **継承プロトコルのレベル**
- 周回数に応じた開始PPと電卓
- 周回ごとの電源・冷却基礎容量ボーナス

### 引き継ぐもの

電源・冷却設備、端末レベル、研究、実績、最適化コア、継承プロトコル、整備資源、自作PC、特別端末、世界プロジェクト、恒久ジョブ効果などは残ります。

### 再構築されるもの

PP、周回処理量、端末台数、タップ強化、ネットワーク構成、実行中ジョブ、未公開のパイプライン、所属組織などはリセットされます。

リセットボタンには今回の獲得量が表示されます。初回は到達直後に実行しても十分な加速を得られ、2周目以降は前回より多くのコアを狙う選択もできます。

## 困ったとき

- **次に何をすればいい？**<br>
  画面上部のNEXT、または「やることリスト」の青く強調された項目を押してください。
- **PPの増え方が急に遅くなった**<br>
  運用タブの電力・温度を確認してください。100%を超えている場合は電源または冷却を増強します。
- **パイプラインが止まる**<br>
  収集・整形・学習・推論のうち、ボトルネック表示に対応する端末を増やします。
- **BGMが鳴らない**<br>
  設定でBGMを有効にし、最初に一度画面をタップしてください。ブラウザの仕様上、操作前には音声を開始できない場合があります。
- **セーブが消えるのが心配**<br>
  設定のエクスポートからJSONファイルを保存してください。インポートすると進行状況を復元できます。
- **最新版にならない**<br>
  ページを一度閉じて開き直してください。Service Workerが新しい版を検出し、自動で切り替えます。

## PWAとしてインストール

- **Android / Chrome**: ブラウザメニューまたはアドレスバーの「アプリをインストール」
- **iPhone / iPad**: Safariの共有メニューから「ホーム画面に追加」
- **PC版Chrome / Edge**: アドレスバーのインストールアイコン

インストール後はホーム画面やデスクトップから起動できます。初回読み込み後は主要ファイルが端末へ保存され、オフラインでもプレイできます。

## セーブデータについて

- 自動保存は初期状態で有効になっており、10秒ごとに保存します。
- 手動保存・読込は設定タブから行えます。
- エクスポートしたJSONはバックアップや別端末への移行に使用できます。
- 「全データを消去」は取り消せません。必要なら先にエクスポートしてください。
